# Garasi Hobby

Aplikasi web internal bengkel untuk mengelola alur servis kendaraan dari kedatangan customer sampai invoice.

Pengguna utama aplikasi ini adalah staff bengkel, seperti Superadmin, Customer Service, Mekanik, QC, dan Kasir.

## Modul Utama

- Dashboard operasional.
- Master data user, role, checklist, material, paket promo, kategori dan item keuangan.
- Operasional pelanggan, order management, pembelian material, dan persediaan material.
- Keuangan untuk account bank/cash, mutasi rekening, transfer saldo, dan input transaksi keuangan.

## Stack

- Laravel
- PostgreSQL
- Spatie Permission
- Metronic UI assets

## Struktur

Project Laravel berada langsung di root repository ini.

```text
D:\garasihobby
```

Bukan lagi di subfolder `app`.
